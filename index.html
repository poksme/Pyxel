Pyxel

A pixel art editor

Pyxel is a PyQt, GameBoy Color inspired, pixel art editor.

To launch Pyxel you will need:

  • Python 2.7
  • PyQt4

Usage:

python pyxel.py

Use the left and right arrows to change color scheme.
